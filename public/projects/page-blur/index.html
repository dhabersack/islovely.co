Page Blur

This extension for Google Chrome is inspired by a video on user interface design published by the Australia-based agency Squareweave.

It blurs the contents of a website so designers can identify which elements of their interfaces stand out and how effective “call to action”-elements are:

You can get Page Blur in the Google Web Store.
